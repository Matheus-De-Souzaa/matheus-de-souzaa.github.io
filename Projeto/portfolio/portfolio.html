Meus conhecimentos & Evolução

Com o Js e meus conhecimentos, aprlicado boas práticas para escrever código limpo e eficiente, e aprendi sobre conceitos mais avançados como callbacks, promessas e programação assíncrona, fundamentais para criar aplicações dinâmicas. Com o uso de ferramentas como o Node.js, me tornei capaz de desenvolver aplicações completas, tanto para o frontend quanto para o backend, aplicando minha capacidade de criar soluções eficazes no mercado.
No React, aprendi a criar interfaces de usuário dinâmicas e eficientes, utilizando componentes que podem ser reutilizados e gerenciados de forma mais organizada. A linguagem permite trabalhar com o estado da aplicação e atualizar a interface de maneira reativa, o que torna o desenvolvimento mais fluido.
No CSS, aprendi a criar designs responsivos e estilos que tornam as interfaces mais agradáveis e funcionais. A linguagem me permite controlar a aparência dos elementos, desde o layout até as cores e animações, sempre com a flexibilidade de adaptar o visual para diferentes dispositivos.
No TypeScript, aprendi a trabalhar com tipagem estática, o que ajuda a evitar erros durante o desenvolvimento. Isso torna o código mais seguro e fácil de manter. Atualmente, estou usando TypeScript para melhorar a qualidade dos meus projetos, aproveitando suas vantagens, como autocompletar e validações mais precisas, o que torna o desenvolvimento mais ágil.
No GitHub, aprendi a gerenciar projetos de forma colaborativa, utilizando controle de versão com Git. Isso facilita o acompanhamento de mudanças no código e a colaboração com outras pessoas. Estou usando o GitHub para armazenar e versionar meus projetos, criando repositórios, fazendo pull requests e mantendo um histórico claro das modificações e eficaz.
No C, aprendi a trabalhar com programação de baixo nível, manipulando diretamente a memória e os recursos do sistema. A linguagem me ensinou a importância de uma estrutura de código bem organizada e o controle rigoroso sobre variáveis e funções.
